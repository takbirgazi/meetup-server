# MeetUP - Video Calling Web Application

**Installation Guide**

* npm install
* Create .env.local file and insert the variables and values
    * PORT
    * DB_USER
    * DB_PASS
    * ACCESS_TOKEN_SECRET
    * MAILGUN_API_KEY
    * MAILGUN_DOMAIN
    * SENDER_EMAIL


**NPM as been used**

- express
- cors
- dotenv
- mongodb
- jsonwebtoken
- nodemon
- mailgun
- bcrypt
