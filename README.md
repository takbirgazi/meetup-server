# MeetUP - Video Calling Web Application

**Installation Guide**

* npm install
* Create .env.local file and insert the variable and values
    * VITE_apiKey
    * VITE_authDomain
    * VITE_projectId
    * VITE_storageBucket
    * VITE_messagingSenderId
    * VITE_appId
    * VITE_API_URL
    * VITE_GEMENI_API_KEY

**NPM as been used**

- express
- cors
- dotenv
- mongodb
- jsonwebtoken
- nodemon
- mailgun
- bcrypt
